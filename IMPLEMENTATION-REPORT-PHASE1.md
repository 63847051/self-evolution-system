## 📝 使用示例

### 示例 1: 简单并行
```
你: @大领导 帮我处理：
  @设计 修改3F更衣室排砖图
  @小蓝 更新这条任务的状态

系统:
  1. 识别到 2 个 @ 提及
  2. 分配给设计专家和小蓝
  3. 并行执行（耗时: max(5分钟, 1分钟) = 5分钟）
  4. 汇总结果
```

### 示例 2: 复杂并行
```
你: @大领导 今天要完成的任务：
  1. @设计 4F会议室合并方案
  2. @技术 写个脚本统计工作时长
  3. @小蓝 记录这些任务到日志

系统:
  1. 识别到 3 个 @ 提及
  2. 分配给设计专家、技术专家、小蓝
  3. 并行执行（耗时: max(5分钟, 3分钟, 1分钟) = 5分钟）
  4. 大领导汇总
```
