## 🎯 功能描述

**核心功能**: 帮助用户发现和安装agent skills

**触发场景**:
- 用户问"how do I do X"
- 用户说"find a skill for X"
- 用户问"is there a skill that can..."
- 用户表达对扩展功能的兴趣

**用途**: 当用户寻找可能存在的可安装技能时使用
